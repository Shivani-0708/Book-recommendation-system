# Book Recommendation System using Python

## Introduction
This project helps users find books according to their preferred genre.

## Features
- Recommend books
- Genre selection
- Simple user interaction

## Technologies Used
- Python
- Lists
- Dictionaries
- Conditional Statements

## How it Works
The user enters a genre and the system recommends books related to that category.

## Conclusion
This project makes book selection easier and demonstrates Python programming concepts.
